## Como Contribuir

Contribuicoes sao bem-vindas! Para contribuir:

1. Faca um Fork do repositorio
2. Crie uma branch de feature (`git checkout -b feature/nova-feature`)
3. Commit suas mudancas (`git commit -m 'Adiciona nova feature'`)
4. Push para a branch (`git push origin feature/nova-feature`)
5. Abra um Pull Request

### Termos de Contribuicao

Ao submeter contribuicoes para este repositorio, voce:

- Cede automaticamente todos os direitos autorais da contribuicao a Pressa Digital
- Garante que possui os direitos necessarios para fazer tal cessao
- Concorda que a contribuicao sera licenciada sob os mesmos termos do projeto
